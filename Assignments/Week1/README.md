# Resume Website 🌐

This is a simple personal resume website built with only HTML. It highlights how basic structure, clean layout, and semantic elements come together on the web.

## 📌 Features

- Personal introduction section  
- Skills list  
- Work experience table  
- Projects showcase  
- Education details  
- Contact information  

---

## 🛠️ Technologies Used

- **HTML5**
